#Sanjeevini Care Hub

## Quickstart

Prerequisites: [Bun](https://bun.sh) (or Node.js 20+).

```bash
bun install      # install dependencies
bun run dev      # start the dev server at http://localhost:8080
```

### Other commands

```bash
bun run build    # production build
bun run start    # run the built app
```

The backend (database, auth) is hosted on Lovable Cloud and pre-configured via `.env` — no local setup needed.
